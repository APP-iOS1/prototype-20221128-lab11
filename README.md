- **🗺️ 지도**
	- 지도 위에 마커 표시
	- 마커를 누르면 해당 장소에서 들었던 노래 확인 가능
- **⭐️ 추천**
	- 반경 500m ~ 6km 범위로 장소 정보 제공
- **✏️ 마이로그**
	- 내 정보 및 노래 기록 열람

## 개발 환경

- xcode 13.0
